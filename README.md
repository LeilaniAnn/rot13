# [Rot13]


Rot13 : A simple web app that allows users to input text and will rotate each letter by 13 places (rot13 encoding)


## Quick start

Several quick start options are available:

* Download python 2.7 (https://www.python.org/downloads/release/python-2711/)
* Clone the repo: `git clone https://github.com/leilaniann/rot13.git`.
* /usr/bin/env python
